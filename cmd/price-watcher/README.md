# WebSocket 价格监控程序

一个最简单的 WebSocket 价格监控程序，用于实时监控 Polymarket UP/DOWN 价格变化。

## 功能特性

- ✅ 启动时显示当前周期信息
- ✅ 自动检测并切换周期（每15分钟）
- ✅ 实时打印 UP/DOWN 价格变化
- ✅ 彩色输出，便于区分 UP（绿色）和 DOWN（红色）

## 使用方法

### 编译

```bash
go build -o price-watcher ./cmd/price-watcher
```

### 运行

```bash
./price-watcher
```

或者直接运行：

```bash
go run ./cmd/price-watcher/main.go
```

### 停止

按 `Ctrl+C` 停止程序。

## 配置

程序会自动读取 `config.yaml` 文件中的代理配置（如果存在）。如果没有配置文件，会使用默认代理 `127.0.0.1:15236`。

## 注意事项

- 程序需要网络连接以访问 RTDS WebSocket 和 Gamma API
- 确保代理配置正确（如果需要）
- 程序会自动处理周期切换，无需手动干预
